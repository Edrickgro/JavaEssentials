Data Structures

Heap : PriorityQueue Class
  • A list or graph/tree
  • Orders elements based on max or min
  • Max: All children are smaller
  • Min: All children are greater
Example 
   Min:    PriorityQueue minHeap = new PriorityQueue();  
   Max:    PriorityQueue minHeap = new PriorityQueue(Comparator.reverseOrder());
Methods 
    offer/add(E e) -> Boolean
    contains(Object o) -> Boolean
    peek() -> head of Queue
    poll() -> head of Queue, removes
    size() -> int
    clear() -> Clears Queue
    toArray() -> Object[]
    iterator() -> Iterator
Complexities
  • O(log n) - > enqueu and dequeu
  • O(n) - > remove() and contains()
  • O(1) - > peek(), element(), size()
Stack
  • (LIFO) - Last In First Out
Example 
   Stack stack = new Stack();
Methods 
    empty() -> Boolean
    peek() -> element at top of list
    pop() -> element at top, removes
    push(E item) -> pushes element to top
    search(Object o) -> index of Object
Complexities
  • O(1) - > push(), pop(), top()
  • O(n) - > search()
Deque: Interface
  • (LIFO) - Last In First Out
  • (FIFO) - First In First Out
Example 
  Deque myDeque = new ArrayDeque();
Methods 
    add() -> Boolean True, tail of deque
    addFirst() -> void, head of deque
    contains(Object o) -> boolean
    peekFirst() -> no removal, null if empty
    peekLast() -> no removal, null if empty
    pollFirst() -> removal, null if empty
    pollLast() -> removal, null if empty
    size() -> int
    removeFirstOccurrence(Object o) -> E
Complexities
  • O(1) - > push(), pop(), top()
  • O(n) - > removeFirstOccurrence()
